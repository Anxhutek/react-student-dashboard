# Premium React Student Dashboard Application 🎓

A premium, state-of-the-art interactive **Student Dashboard** designed using React.js and Vite. Tailored for academic tracking, section notifications, and test preparations.

---

## 🏫 Project Overview
Developed by **Anshu Kumar Verma** (B.Tech 1st Year, 2nd Semester, CSE-19 at ABES Engineering College), this application integrates:
- **Interactive Sidebar Navigation**: Seamless switching between tabs (Dashboard, Test Preparation, Profile).
- **Core Dashboard Metrics**: Displays dynamic pending tasks, target grades, and finished practical experiments.
- **Interactive Quiz Engine**: Built-in test-preparation module to practice JavaScript, React hooks, and REST controllers.
- **Modern Dark Glassmorphic Design**: UsesOutfit typography, smooth gradients, radial background reflections, and clean responsive CSS cards.

---

## 🚀 How to Run Locally

### Prerequisites
Make sure you have **Node.js** installed on your system.

### Steps
1. Navigate to the project root directory:
   ```bash
   cd react-student-dashboard
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open the displayed URL (typically `http://localhost:5173`) in your browser.
